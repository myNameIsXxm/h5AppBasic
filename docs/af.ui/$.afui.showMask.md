#$.afui.showMask(text, value);

```

Show the loading mask with specificed text

```

##Example

```
 $.afui.showMask()
 $.afui.showMask('Fetching data...')
 $.afui.showMask('Fetching data...', 20000) //number of milliseconds to display the mask, default 15 secs
 
```


##Parameters

```
text                          string=
value                         number=

```

##Returns

```
undefined
```
